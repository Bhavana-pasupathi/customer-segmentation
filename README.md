# Customer Segmentation using K-Means Clustering

## 📌 Project Overview

Customer Segmentation is a machine learning project that groups customers into different categories based on their spending behavior and annual income. This helps businesses understand customer patterns and create targeted marketing strategies.

## 🎯 Objectives

* Analyze customer data
* Identify customer groups with similar behavior
* Improve business decision-making through data-driven insights
* Visualize customer segments using clustering techniques

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Machine Learning Algorithm

* K-Means Clustering
* Elbow Method for optimal cluster selection

## 🔄 Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Finding Optimal Clusters using Elbow Method
5. Applying K-Means Clustering
6. Data Visualization
7. Customer Segment Analysis

## 📈 Results

The project successfully segments customers into different groups based on spending score and annual income. These insights can help businesses develop personalized marketing strategies and improve customer engagement.
